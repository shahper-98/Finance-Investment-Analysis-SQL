# Finance Investment Analysis using SQL

## Project Overview

This project analyses investor behaviour, savings objectives, and investment preferences using SQL Server. The dataset contains information about investors' demographics, preferred investment avenues, investment objectives, monitoring habits, and expected returns.

The project demonstrates SQL skills ranging from basic data exploration to advanced analytical techniques.

---

## Objectives

* Analyse investor demographics.
* Identify popular investment avenues.
* Study savings objectives and investment goals.
* Understand factors influencing investment decisions.
* Perform intermediate and advanced SQL analysis.

---

## Dataset

The dataset contains information about:

* Gender
* Age
* Investment avenues
* Mutual funds
* Equity market
* Government bonds
* Fixed deposits
* Gold investments
* Investment objectives
* Investment duration
* Savings objectives
* Reasons for investment choices
* Investment monitoring habits
* Sources of investment information
